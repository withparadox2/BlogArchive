Adding a button to media upload area/TinyMCE editor, than you can search and insert Xiami music into your posts easily.

== Description ==

TinyMCE Xiami Music makes searching and inserting Xiami music into your posts easily.<br />
If you are using Windows Live Writer, then I recommend you trying my [WLW Xiami Music](http://peema.us/archives/wlw-xiami-music.html).<br />
TinyMCE Xiami Music让你很容易的在后台搜索并在文章中插入虾米音乐。<br />
如果你正在使用Windows Live Writer，那么建议你尝试我的[WLW Xiami Music](http://peema.us/archives/wlw-xiami-music.html)。

* For MORE information, please visit [Plugin official page](http://peema.us/archives/tinymce-xiami-music.html).
* 了解详情请访问[插件官方页面](http://peema.us/archives/tinymce-xiami-music.html)。

== Installation ==

1. Install this plugin by upload or search & install.
2. Activate it in `Plugins` menu.
3. No settings and no database access needed.
1. 上传安装或搜索`TinyMCE Xiami Music`自动安装。
2. 在`插件`菜单中启动。
3. 本插件无设置，不写数据库。
